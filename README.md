# 抗议专版，助力抗议信息收集统计

## 简介

使用wechaty机器人+维格表 收集防疫群统计信息，可运行与任何windows环境，登陆个人wechat账号


## 快速入门

1. 到 [维格表官网](https://vika.cn/) 注册维格表账号

2. 创建一个名字为 mp-chatbot 的空间

3. 获取维格表token

![vika](./doc/images/vika_token.png)

4. 替换ding-dong-bot.ts文件中的vika-token

5. 运行 

```
npm run start
```
